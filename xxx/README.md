# キリスト・プログラム

悩みや不安を入力すると、**聖書の引用**とともに解決のヒントを返すAIチャットアプリです。

## 機能

- **チャット形式**で悩みを入力
- **聖書の言葉**（書名・章・節）を引用して回答
- **デモモード**：APIキーなしでも、キーワードに応じた聖書の引用で応答
- **AIモード**（任意）：OpenAI APIキーを設定すると、より自然な対話と聖書に基づく助言が可能

## 使い方

1. `index.html` をブラウザで開く
2. 悩みや質問を入力して「送信」
3. 聖書の引用と短い助言が表示されます

### APIキーでより自然な対話にする（任意）

1. 画面右上の ⚙️ をクリック
2. 「OpenAI API キー」に `sk-...` のキーを入力して「保存」
3. 以降はAIが聖書を引用しながら、悩みに合わせた回答を生成します

※ APIキーはブラウザのローカルストレージにのみ保存され、サーバーには送信されません。

## ファイル構成

- `index.html` — チャットUI・設定モーダル
- `style.css` — レイアウト・デザイン
- `app.js` — チャット処理・デモ用聖書引用・OpenAI連携

## 技術

- フロントのみ（HTML / CSS / JavaScript）
- デモ時はキーワードに応じた聖書の引用を表示
- AI利用時は OpenAI Chat Completions API（`gpt-4o-mini`）で聖書に基づく回答を生成
